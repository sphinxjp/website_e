Create Project
==================

To create document by Sphinx, you need to create the project of the
document. Creating project is similar about Scaffold of `Ruby on Rails
<http://rubyonrails.org/>`_ or `Django <http://djangoproject.jp/>`_
Project, only answer some question.

This chapter describes the document tree structure and accomplish the
form.
